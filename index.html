Machine Learning Projects

Two applied ML projects from coursework — each presented with problem statement, methodology, results, and key observations.

01
CNN Computer Vision Transfer Learning

Image Classification Using Convolutional Neural Networks

Problem

Designing a system to classify images into predefined categories — with applications in medical imaging and autonomous systems.

Results

94.2% classification accuracy on the test dataset — a 12% improvement over baseline, with precision and recall balanced across all classes.

Evaluation Metrics

MetricScore
Accuracy94.2%
Precision93.8%
Recall94.1%
F1-Score93.9%
Methodology & Key Observations

Methodology

Transfer learning with a pretrained ResNet-50 backbone. Key steps included:

  • Applying data augmentation techniques to improve generalization
  • Fine-tuning classification layers on the target dataset
  • Implementing a structured training pipeline with validation monitoring

Key Observations

  • Transfer learning significantly reduced training time while maintaining strong predictive performance
  • Data augmentation proved essential for preventing overfitting when training data was limited
  • Evaluation metrics beyond accuracy were necessary to properly assess model behavior
02
LSTM Sequence Modeling RNN

Sequence Modeling with Recurrent Neural Networks

Problem

Modeling temporal patterns in sequential data — critical for applications in natural language processing and time-series prediction.

Results

89% validation accuracy, with the bidirectional LSTM architecture outperforming the standard variant by 6%.

Evaluation Metrics

MetricScore
Validation Accuracy89.0%

Methodology & Key Observations

Methodology

LSTM-based RNN architecture. The training process included:

  • Use of pretrained embeddings
  • Dropout regularization to reduce overfitting
  • Comparison between standard and bidirectional LSTM architectures

Key Observations

  • Careful data preprocessing had a significant impact on model performance
  • Hyperparameter tuning required systematic experimentation rather than heuristic selection
  • Bidirectional architectures improved contextual understanding in sequential tasks
